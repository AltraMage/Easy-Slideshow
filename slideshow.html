About

I couldn't find an easy html slideshow that worked with N amount of slides, so I made a template that can use any amount from 1 to any amount.

FYI, you can have whatever text you want in the buttons, it autosets to the arrow chars in the JS

Change the button design in the sampled css, the .slideshow button refers only to the nagivation buttons for the slides.

All images should be the same size, is a little funky if they aren't. At the very least could they be similar height, please?

Usage

Step 1. Import JS / css (css is optional but aligns the buttons) Using the script & link tags respectively. N.B.Javascript tag should have defer after the src.

Javascript CSS

Step 2. Create a div with a class named slideshow (class="slideshow"), this will be your container for the css.

Step 3. Add buttons x2, One with id="left", other with id="right". Those are your navigation buttons.

Step 4. Add slides & content. Each slide can be its own div or image. Link these elements with class="slide".

If everything has gone properly you should now have a working slideshow.

Troubleshooting

Make sure the Javascript and CSS are in the same directory as the webpage; not just linked.
